# learning-platform

Plateforme de cours en ligne développée avec Next.js 14.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Configuration

1. Copiez `.env.example` vers `.env.local`
2. Configurez vos variables d'environnement
3. Initialisez la base de données avec Prisma

```bash
npx prisma generate
npx prisma db push
```
